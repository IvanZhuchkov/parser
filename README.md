# parser
Данное приложение распознаёт текст на русском и английском языках, а так же написанные на русском, но английской раскладкой и напоминает о событиях. Все испытания на данных фразах прошли успешно!
Он понимает фразы типа: "Поесть в 11 часов завтра.", " Тренировка 23.09.2023.", " Выключить плиту через 5 минут.", "Go shopping on friday", "Gjuekznm c cj,frjq d gznm xfcjd" и тд.
